# sjc-apps

Kleine oefen-webapps, gebundeld in één repository met een gedeelde stijl.

- [`index.html`](index.html) — landingspagina met links naar de apps
- [`shared/base.css`](shared/base.css) — gedeelde kleuren, typografie en componenten (kaarten, knoppen, hud, ...)
- [`kwadratenjacht/`](kwadratenjacht/) — zoek zo snel mogelijk alle volkomen kwadraten, samen met een tweede speler
- [`oversteek/`](oversteek/) — steek de rivier over door sommen met negatieve getallen op te lossen
- [`pariteitscontrole/`](pariteitscontrole/) — instelbaar rooster om pariteitsbits te oefenen

Elke app heeft een eigen `style.css` die alleen de app-specifieke opmaak bevat; de rest komt uit `shared/base.css`.

Gehost via GitHub Pages vanaf de `main`-branch.
